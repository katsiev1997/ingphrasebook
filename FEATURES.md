# 🚀 Рекомендуемые функции для реализации

Документ содержит список рекомендуемых функций для улучшения приложения IngPhrase и повышения эффективности изучения ингушского языка.

## 🎯 Приоритетные функции (рекомендуется начать с них)

### 1. **Система повторений (Spaced Repetition)**

**Описание:**

- Алгоритм интервальных повторений (Anki-подобный)
- Автоматическое планирование повторений на основе успешности ответов
- Уровни изученности фраз (новые → изучаемые → изученные → освоенные)
- Уведомления о необходимости повторения
- Настройка интервалов повторений

**Технические детали:**

- Новая таблица `phraseLearningProgress` в БД
- Поля: `userId`, `phraseId`, `level`, `nextReviewDate`, `reviewCount`, `successRate`
- Алгоритм SM-2 или упрощенная версия
- API endpoints для обновления прогресса

**Польза:** Эффективное запоминание фраз по научно обоснованному методу

---

### 2. **Флеш-карточки (Flashcards)**

**Описание:**

- Режим карточек с переворачиванием (ингушский ↔ русский)
- Настройка направления изучения
- Фильтрация по категориям и уровню изученности
- Автопереход после ответа
- Режим "знаю/не знаю" с автоматическим планированием

**Технические детали:**

- Новый роут `/flashcards`
- Компонент `FlashcardView` с анимацией переворота
- Интеграция с системой повторений

**Польза:** Быстрая и интуитивная тренировка запоминания фраз

---

### 3. **Детальная статистика изучения**

**Описание:**

- Прогресс по категориям (сколько изучено в каждой категории)
- График активности (дни недели, время суток)
- История изучения фраз с датами
- Тепловая карта активности (календарь GitHub-style)
- Стрики (дни подряд изучения)
- Общее время, потраченное на изучение

**Технические детали:**

- Таблица `userActivity` для отслеживания активности
- Новый роут `/statistics`
- Графики с использованием Recharts
- API для получения детальной статистики

**Польза:** Мотивация через визуализацию прогресса и отслеживание достижений

---

### 4. **Режим диктовки**

**Описание:**

- Аудио диктант: прослушать фразу и написать её
- Проверка правильности написания
- Подсказки при ошибках
- Статистика по диктовкам
- Разные уровни сложности

**Технические детали:**

- Новый роут `/dictation`
- Компонент для записи текста
- Алгоритм сравнения текста (с учетом вариантов написания)
- Интеграция с Web Speech API для голосового ввода (опционально)

**Польза:** Тренировка письма и восприятия на слух

---

### 5. **История и закладки**

**Описание:**

- История просмотренных фраз
- Закладки с заметками к фразам
- Теги для организации фраз
- Поиск по заметкам и тегам
- Экспорт закладок

**Технические детали:**

- Таблица `phraseBookmarks` (userId, phraseId, notes, tags, createdAt)
- Таблица `phraseHistory` (userId, phraseId, viewedAt)
- Новый роут `/bookmarks`
- Компонент для управления тегами

**Польза:** Удобная организация изученных материалов

**Описание:**

### 10. **Экспорт и импорт данных**

**Описание:**

- Экспорт избранных фраз в PDF/CSV
- Импорт собственных фраз
- Печать разговорника
- Офлайн-режим с синхронизацией
- Резервное копирование данных

**Технические детали:**

- Библиотека для генерации PDF (jsPDF или react-pdf)

**Польза:** Работа без интернета и сохранение данных

---

### 11. **Расширенный игровой режим**

**Описание:**

- Режим "Скорость": ответы на время
- Режим "Выживание": игра до первой ошибки
- Режим "Категория": только по выбранной категории
- Мультиплеер (если будет несколько пользователей)
- Ежедневные челленджи

**Технические детали:**

- Расширение существующего компонента `QuizGame`
- Новые типы игровых сессий
- Таймер для режима скорости
- API для мультиплеера (WebSockets или Server-Sent Events)

**Польза:** Разнообразие тренировок и повышение вовлеченности

---

### 12. **Аналитика ошибок**

**Описание:**

- Топ ошибок пользователя
- Сложные фразы (часто неправильные)
- Рекомендации на основе ошибок
- Визуализация проблемных зон

**Технические детали:**

- Таблица `userMistakes` (userId, phraseId, mistakeCount, lastMistakeDate)
- API для отслеживания ошибок
- Компонент аналитики ошибок
- Интеграция с игровым режимом

**Польза:** Фокус на проблемных местах для эффективного обучения

---
