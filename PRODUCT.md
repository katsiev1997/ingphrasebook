# IngPhrase — Product One-Pager

## Problem

Russian speakers who want to learn or refresh Ingush can look up phrases, but the app does not help them remember or return regularly. Today it is a phrasebook + quiz, not a practice habit.

## ICP

- Primary: Russian-speaking learners of Ingush (diaspora, beginners, heritage speakers)
- Secondary: people who need a quick phrase reference
- Contributors: moderators/admins who curate phrases and audio

## Goal (this quarter)

Turn IngPhrase into a product with a **regular practice loop**: open → review due cards → come back tomorrow.

## North-star metrics

1. Share of logged-in users with ≥1 review session per week
2. Secondary: D7 retention

Supporting: reviews/day, streak length, quiz → SRS enqueue rate

## Now (committed)

1. Flashcards + simplified SM-2 SRS
2. Habit layer: activity log, streak, daily goal (10 reviews), soft browser notifications
3. Dialogues UI, thin learning stats, dictation v1 (phrases with audio)

## Non-goals (this quarter)

- Social features, multiplayer, forums
- ML recommendations
- Grammar CMS
- AI pronunciation scoring
- Full PWA/offline sync
- In-app survey product (use external form if needed)
- Heavy analytics dashboards / heat maps

## Baseline checklist

Capture before / after Phase 1:

- [ ] Registered users count
- [ ] Quiz completion rate
- [ ] Favorites usage
- [ ] Return visits (Yandex Metrika)
- [ ] Content audit: phrases total, % with audio, empty categories

## Success for Phase 1 MVP

Logged-in user completes a flashcard session; due cards reappear on schedule; Study hub shows progress; guests can try a demo without persistence.
